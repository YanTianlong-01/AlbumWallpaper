# AlbumWallpaper

## 功能

AlbumWallpaper是一款基于python开发的，跨平台的专辑墙壁纸生成器。


- 输入歌单链接，自动批量下载歌单中每一首歌曲的封面（目前仅支持网易云音乐）。
- 根据文件夹中的所有图片生成一个专辑墙封面。
- 可以自己设定图片，不限定于歌曲封面，只需将图片存放在指定文件夹下即可。
- 可以自己调整生成壁纸的分辨率、长宽比。
